3. มีการสร้างหน้าแสดงรายละเอียดข้อมูล เมื่อกดมาจากหน้ารายการ (ตามโจทย์ข้อ 3)
4. รองรับการแสดงผลได้ทั้งแนวนอนและแนวตั้ง (ตามโจทย์ข้อ 4)
